01 cat about.md

I'm a full-stack developer building AI-powered products from the ground up — right now that means an EdTech SaaS platform and a file-automation tool, both built on Flask and wired into LLM APIs for the parts that actually need intelligence. I like getting close to the machine: I spend time in Linux internals, troubleshooting systems at the driver and display-manager level, and studying offensive and defensive security tooling to understand how the software I ship can be attacked and defended.

My approach is hands-on and end-to-end — architecture, auth, payments, deployment, and the AI features that tie it together — rather than sticking to one layer of the stack.

Python / Flask LLM Integration SQL & ORM Design Linux Systems Security Fundamentals
